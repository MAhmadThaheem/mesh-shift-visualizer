# Mesh Circular Shift Visualizer

A full-stack interactive web application that simulates and visualizes circular q-shift operations on a 2D mesh topology.

## Prerequisites
- Node.js (v18+)
- npm or yarn

## Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies (React, Framer Motion, Lucide React).
3. Run `npm run dev` to start the local development server via Vite.

## Algorithms Implemented
- **Stage 1 — Row Shift:**  (`q mod √p`)
- **Stage 2 — Column Shift:** (`⌊ q / √p ⌋`)

Enjoy visualizing parallel algorithms!
